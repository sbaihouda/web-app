# 

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Docker build
```
docker build -t lpo-app .
```

### Docker run (localhost)
```
docker run -it -p 8080:80 --rm --name App-lpo lpo-app
```
